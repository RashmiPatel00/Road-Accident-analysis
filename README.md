# Road Accident Analysis

## Overview

This project analyzes road accident data to identify patterns, trends, and key risk factors contributing to accidents. The analysis focuses on deriving insights that can support better decision-making and improve road safety.

---

## Objectives

* Analyze accident data to identify major causes
* Identify high-risk locations and time periods
* Study the impact of external factors such as weather and vehicle types
* Provide actionable insights using data visualization

---

## Tech Stack

* SQL (data cleaning, transformation, and querying)
* Power BI (data visualization and dashboard creation)

---

## Dataset

* Source: (online free platform)
* Features include:

  * Date and time of accident
  * Location
  * Vehicle type
  * Weather conditions
  * Number of casualties and severity

---

## Data Processing

* Cleaned and transformed raw data using SQL
* Performed joins, aggregations, and filtering to prepare analysis-ready datasets
* Structured data for efficient reporting and visualization

---

## Analysis Performed

* Time-based analysis (monthly and hourly trends)
* Location-based accident distribution
* Vehicle type involvement
* Weather impact on accidents
* Severity analysis (fatal and non-fatal accidents)

---

## Dashboard

* Built an interactive Power BI dashboard to visualize accident trends
* Included filters for location, time, and vehicle type
* Enabled drill-down analysis for deeper insights

---

## Key Insights

* Accident frequency is higher during peak traffic hours
* Certain locations consistently show higher accident rates
* Weather conditions such as rain contribute to increased accidents
* Two-wheelers are among the most affected vehicle type
